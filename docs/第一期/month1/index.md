## 📅 详细学习路线与任务

### 第一周：坐标系与位姿变换

- **学习目标**：理解机器人运动的空间表示，掌握齐次变换矩阵的使用。
- **核心内容**：
  1. 笛卡尔坐标系与关节坐标系
  2. 齐次矩阵（4×4）表示位姿
  3. 平移、旋转（欧拉角、四元数）
  4. 坐标系之间的转换与链式计算
- **学习资料**：`机器人基础——坐标系与位姿变换(1).docx`
- **本周作业**（输出 PDF 文档）：
  1. 为什么需要用一个整体的"齐次矩阵"来表示位姿，而不是分开用旋转和位移？
  2. 如果一个点在 A 坐标系下有坐标，我们如何把它转换到 B 坐标系？
  3. 多个关节坐标系之间的变换，最后是如何得到末端相对基座的位姿的？

### 第二周：轨迹规划

- **学习目标**：理解机器人从起点到终点的运动方式和路径生成。
- **核心内容**：
  1. 插值方法（直线插值、多项式插值）
  2. 笛卡尔空间轨迹 vs 关节空间轨迹
  3. 时间参数化与速度/加速度约束
  4. 基本规划算法（RRT、PRM 简介）
- **学习资料**：`具身智能基础——机器人轨迹规划.docx`
- **本周作业**（输出 PDF 文档）：
  1. 轨迹规划时，为什么要限制速度和加速度？
  2. 如果规定两秒钟必须完成一次动作，应该如何安排轨迹？

### 第三周：机器人运动学

- **学习目标**：建立关节运动与末端位姿的数学关系，理解 IK/FK。
- **核心内容**：
  1. FK（正运动学）：已知关节角 → 计算末端位姿
  2. IK（逆运动学）：已知末端位姿 → 计算关节角
  3. 奇异性与冗余度问题
- **学习资料**：
  - 快速了解 IK/FK（7分钟短片）：[BiliBili链接]
  - 简易版原理过关：[CSDN链接]
  - FK 详细视频课：[FK(1)链接], [FK(2)链接]
  - IK 详细视频课：[引言链接], [IK(1)链接], [IK(2)链接]
  - 理论教材：IK 理论书籍、《机器人学导论（原书第3版）.pdf》
- **本周作业**（输出 PDF 文档）：
  1. 正运动学和逆运动学的输入和输出是什么？
  2. 一个平面两关节机械臂去到同一个目标点，可能会有几种解？为什么？
  3. 如果目标点超出了机械臂的最大工作范围，会怎样？

### 第四周：深度强化学习

- **学习目标**：掌握强化学习基础概念、学会结合强化学习与深度学习。
- **核心内容**：
  1. 强化学习的基本框架（状态、动作、奖励、策略、价值函数）
  2. 策略梯度方法：REINFORCE、Actor-Critic
  3. 进阶算法：PPO、A3C
  4. 强化学习与深度学习结合（Deep RL）
- **学习资料**：
  - `具身智能基础——强化学习.docx`
  - 视频课：[【2025最新版】王树森深度强化学习全套课程]
- **本周作业**（输出 PDF 文档）：
  1. 强化学习的目标是什么？
  2. DQN 和 Q-learning 的区别是什么？
  3. 为什么 PPO 在实际中比 REINFORCE 更稳定？
  4. A3C 的优势是什么？
