# 谋先飞线上实习：机器人仿真实训（🧪 Beta公测版）

> [!WARNING]  
> 🧪 **Beta公测版本提示**：本项目教程主体（第一、二个月大纲及部分核心内容）已完成构建，正在持续优化学习资料与作业细节，欢迎大家提 Issue 反馈问题或参与共建。

本项目是“谋先飞线上实习”的核心实训内容，采用开源社区共建模式。旨在帮助学习者打通**从传统机器人学到前沿具身智能**的技术链路。通过两个月的系统学习，你将掌握正逆运动学原理与深度强化学习算法，并最终在真实物理仿真引擎（IsaacLab / MotrixLab）中完成四足机器人的导航任务迁移与控制。

## 🎯 项目受众与前置要求

* **面向群体**：参与“谋先飞线上实习”的同学，以及对具身智能、机械臂控制、强化学习与物理仿真感兴趣的开源学习者。
* **我们能带来什么**：提供一条“理论+实操”的完整闭环路线，带你手把手拆解 IsaacLab 环境配置，并落地到实际的工程迁移任务中。
* **基础要求**：具备 Python 编程基础，了解基本的线性代数（矩阵运算）与微积分知识。
